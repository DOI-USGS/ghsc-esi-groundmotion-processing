gmprocess.io.utils¶

gmprocess.io.utils.flatten_directory(directory)[source]¶

Prepare a messy directory to be read in.

This is largley motivated by how CESMD distributes data with randomly zipped files and subdirectories. This flattens the directory structure and prepares it for use with either the gminfo command line program or to be read in with teh directory_to_streams method.

Parameters

directory (str) – Directory of ground motion files (streams).

Returns

None.

Checks whether times are evenly spaced.

Parameters

  • times (array) – Array of floats of times in seconds.

  • rtol (float) – The relative tolerance parameter. See numpy.allclose.

  • atol (float) – The absolute tolerance parameter. See numpy.allclose.

Returns

True if times are evenly spaced. False otherwise.

bool
